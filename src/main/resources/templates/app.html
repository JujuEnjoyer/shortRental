Два блока в один день — обычно две разные брони (пересечение по датам: например выезд одного и заезд другого, или два импорта ICS). Подпись → заезд — первый день брони, ← выезд — последний. Цвет полоски слева различает брони. Имя в ячейке берётся из профиля гостя в системе; импорт ICS часто даёт только даты — тогда видно «без профиля», пока не создадите гостя и не привяжете бронь.

        Каналы сообщений

        Подключите площадку к объекту, чтобы читать диалоги и отправлять ответы через общий чат.

        Avito OPEN_API использует client_id и client_secret; user_id определяется автоматически.
        Циан OPEN_API использует ACCESS KEY; при необходимости можно указать employeeId.
        Партнерские API Hostaway, Channex, Booking.com и Guesty работают через API token, если он выдан пользователю.
        Fallback-каналы Для Яндекс Путешествий, Островка и Суточно.ру доступны mock и резервные режимы для демонстрации.
