# Чеклист демонстрации ShortRental

## Перед показом

1. Запустить приложение:

```bash
./mvnw spring-boot:run
```

2. Открыть:

```text
http://localhost:8080
```

3. Зарегистрировать или войти под арендодателем.

## Основной сценарий

1. Открыть "Настройки" -> "Общие".
2. Нажать "Подготовить демо-сценарий".
3. Скопировать гостевую ссылку.
4. Открыть ссылку в режиме инкогнито.
5. Заполнить тестовые паспортные данные.
6. Загрузить тестовые изображения.
7. Вернуться в кабинет арендодателя.
8. Открыть "Документы" -> "На проверку".
9. Проверить данные гостя.
10. Подтвердить документ и сформировать PDF.

## Что показать комиссии

- календарь и бронирования;
- единый чат и шаблоны ответов;
- документ и гостевую ссылку;
- защищенную форму гостя;
- проверку паспортных данных;
- аналитику;
- настройки интеграций;
- вкладку "Безопасность" с политикой хранения и журналом действий.

## Что не делать на демонстрации

- не использовать настоящие паспортные данные;
- не писать настоящим гостям на внешних площадках;
- не вводить реальные API-ключи в публичной демонстрационной среде;
- не удалять данные перед показом, если нужно демонстрировать готовый сценарий.
